Detailed Description

The DeclarationName class.

The base class for all names of declarations. The name of a declaration may exactly correspond to an identifier in the syntax or it may be something else, encompassing certain semantics of the declaration in question, e.g., a type declaration for which no identifier is provided if of name Anonymous.

Note
Similar class in LLVM/clang: clang::DeclarationName.
