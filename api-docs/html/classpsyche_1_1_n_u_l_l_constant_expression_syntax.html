Detailed Description

The NULLConstantExpressionSyntax class.

Attention
This SyntaxNode is produced if NULL is configured to be treated as a builtin through a language extension, prior to parsing. The grammar extension applies on 6.4.4 and, consequently, on 6.5.1 as well.
